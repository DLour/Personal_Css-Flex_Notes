just learning css flex/flexbox

will be using display:flex for every container
below justify-content:flex-end <= pushes content right.

Justify-Content:space-evenly

I use justify-content: space-evenly; <= this makes even space between all the elements within the yellow container.
Even space is made between the yellow container, and the other items within the containers.

Justify-Content:space-between

only makes space between the other boxe's, instead of container
when it has no more space in the container, it sticks to the wall of it.

Align-content:Center

The center seems to align the item VERTICALLY CENTER from starting position.

flex-direction:column

Takes the last two block, and align it to the bottom of the red box
while keeping its original order.

Note: When Using column-reverse, it just switches the last block with first block. this then reverse's the order of the boxes! this is not shown here.

flex-direction:ROW-reverse

**Not using regular row

SImilar to justify-content:flex-end, BUT, when using row-reverse, it switches the order
just like column-reverse, and it will move to the right side of the container, while changing orders!
